# rtn-turbo-geocode

A Turbo Native Module to Access native geocode APIs in React Native.

## Installation

```sh
npm install rtn-turbo-geocode
```

## Usage


```js
import { multiply } from 'rtn-turbo-geocode';

// ...

const result = multiply(3, 7);
```
